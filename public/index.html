Optimiza tu proyecto Vite

Selecciona un archivo ZIP de tu proyecto Vite y genera una versión con pre‑rendering y SEO lista para producción.

⚠️ Warning: This will modify files in the specified folder directly. Ensure you have a backup or git commit.

Siguientes pasos

  • Sube el ZIP generado a GitHub o Bitbucket para gestionar el código fuente.
  • Conecta tu repositorio a un servicio de despliegue estático como Vercel o Netlify para publicar tu web.
  • Si prefieres compilar localmente, descomprime el ZIP y ejecuta npm install y npm run build.
  • Consulta la documentación de tu plataforma de despliegue para asociar un dominio personalizado.
